# 大纲工具文档

## 模块入口

- `脚本/大纲工具/outlinerX_v1.11.py`

## 功能职责

大纲工具提供一个可停靠的 Maya Outliner 扩展面板，用于快速整理场景层级、裁剪显示范围、排序节点、设置大纲颜色标记，并通过内嵌 QShade 面板浏览和指定材质。

## 主要能力

- 层级筛选：可按定位器、关节、网格、曲线、曲面、摄像机、灯光、组和相似模型选择层级。
- 大纲整理：支持仅显示所选、仅显示已标色、显示全部、展开所选一级、节点上移下移和层级排序。
- 颜色标记：支持对大纲、视图线框和材质进行颜色标记，并可重置颜色标记。
- 材质面板：QShade 支持材质缩略图浏览、指定材质、克隆材质、删除未使用材质和刷新网格。
- 快速备注：可创建备注节点，便于在场景层级中放置书签式标记。

## 使用方式

在 Maya 脚本编辑器中切换到 Python 标签页，复制并运行 `outlinerX_v1.11.py`。运行后会创建 `Outliner X 大纲工具` 可停靠面板。

## 维护边界

- 用户可见文案、警告信息、工具提示和说明性注释可按简体中文维护。
- Maya 命令参数、函数名、内部控件标识、节点类型名和材质节点名不应随意改名，避免破坏脚本运行逻辑。
- QShade 相关尺寸、滚动和列数计算逻辑与 Maya UI 状态耦合较高，调整前应先在 Maya 内实际验证。

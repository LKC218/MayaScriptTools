# 项目代码地图

## 根目录

- `README.md`：项目总说明和基础使用入口。
- `.gitignore`：本地缓存、虚拟环境、编辑器配置和临时文件忽略规则。
- `docs/`：脚本规范、工具说明和模块维护文档。
- `docs/推送工作汇总-260511.md`：2026-05-04 至 2026-05-11 的 LKC218 推送工作汇总。
- `脚本/`：Maya Python 脚本和插件源码。
- `生图提示词/`：提示词资料。
- `output/`：本地输出目录。

## 脚本目录

- `脚本/相机工具/`：相机对齐相关 Maya 脚本。
- `脚本/大纲工具/`：Maya 大纲管理、层级整理、颜色标记和材质快速指定工具，入口为 `outlinerX_v1.11.py`。
- `脚本/建模工具/`：建模、修改器、投影和导入导出相关 Maya 脚本。
- `脚本/建模工具/intersectionSolver/`：模型穿插检测与辅助修复工具，入口为 `intersectionSolver.py`。
- `脚本/建模工具/小学智修改器历史窗口V14/`：修改器历史窗口旧版本脚本，入口为 `小学智修改器历史窗口 V14.py`。
- `脚本/建模工具/小学智修改器历史窗口16/`：修改器历史窗口当前版本脚本，入口为 `小学智修改器历史窗口16.py`。
- `脚本/建模工具/小学智晶格分段修改器V2.3/`：晶格分段修改器脚本，入口为 `小学智晶格分段修改器 V2.3.py`。
- `脚本/建模工具/组件投影插件0.2/`：组件投影插件脚本，入口为 `组件投影插件0.2.py`。
- `脚本/建模工具/glb工具maya/`：Maya GLB 导入导出插件，入口为 `install.py` 和 `maya_glb_tool.py`。

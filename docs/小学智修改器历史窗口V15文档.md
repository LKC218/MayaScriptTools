# 小学智修改器历史窗口 V15 文档

> 适用范围：Maya 建模流程 / 多边形历史管理 / 材质与清理辅助  
> 当前脚本：`脚本/建模工具/小学智修改器历史窗口16.py`  
> 当前版本：`V15`  
> 文档整理日期：2026-04-28

---

## 1. 工具概述

小学智修改器历史窗口 V15 是一个面向 Maya 建模场景的综合工具窗口。它把“历史节点查看与操作”作为核心，同时整合了材质复制、一键法线、UV 传递、随机材质 ID、场景清理、贴图检查、无光照视图和智能选内壳等高频建模辅助功能。

这个工具的定位不是单一命令，而是一个围绕建模修改历史展开的集成工作台，适合在处理中低模编辑、历史复用、材质检查、导出前清理时持续使用。

---

## 2. 核心能力

### 2.1 历史堆栈可视化

- 显示当前选中模型的历史节点列表
- 支持开启/关闭节点生效状态
- 支持删除历史节点
- 支持节点右键菜单操作
- 支持 Shift 连选、Ctrl 加选
- 支持复制一组历史参数并粘贴到其他模型

### 2.2 Mini AE 参数面板

- 点击历史节点后，在窗口下半区自动生成参数控件
- 自动读取可调属性和通道盒属性
- 过滤底层系统属性，减少无关干扰
- 不再额外弹窗，适合连续调参

### 2.3 顶部对象命名区

- 单击查看当前对象
- 双击进入重命名模式
- 单选时支持普通重命名
- 多选时支持批量命名
- 批量命名格式为：`基础名_01`、`基础名_02`、`基础名_03`

### 2.4 建模与材质辅助工具

- 独立复制并保留历史
- 批量拓扑传 UV
- 复制材质、关联粘贴、独立粘贴
- 一键为材质连接法线贴图
- 按 UV 壳生成随机材质 ID
- 一键应用检查贴图并旋转贴图
- 视图切换到无光照 Flat 模式

### 2.5 场景清理与结构整理

- 清理未使用材质和无用节点
- 删除空组与无效网格
- 删除用户命名空间
- 智能选出被包裹在外壳内部的零件

UI 主体由三部分组成：

### 3.1 顶部标题区

- 左侧：帮助按钮

- 右侧：刷新按钮

### 3.2 中部工具栏

按功能分为多排按钮：

| 工具分组 | 功能 |

| 模型与 UV | 独立复制(带历史)、批量拓扑传 UV |
| 材质管理 | 复制材质、关联粘贴、独立粘贴、一键法线 |
| 优化清理 | 随机ID、清理未用、删空组、删命名空间 |
| 选择辅助 | 智能选内壳 |
| 贴图检查 | Map 2、贴图 +45°、贴图 -45°、无光照 Flat |

- 上部分：历史列表
- 下部分：Mini AE 参数面板
- 中间可拖拽分隔条调整比例
- 当前版本已支持窗口自由缩放

## 4. 使用流程

### 4.1 查看和管理历史

1. 在场景中选择一个模型
2. 窗口顶部会显示当前对象名称
3. 下方历史区会列出相关历史节点
4. 点击节点可以在下方面板直接修改参数
5. 勾选框可开关该节点是否生效
6. 右键节点可复制、删除、重命名或粘贴历史组合

### 4.2 复制并复用历史步骤

1. 选择一个模型并在历史列表中选中一个或多个历史节点
2. 右键执行“复制选中项参数”
3. 切换到目标模型或目标组件
4. 右键空白区或节点菜单执行“粘贴历史组合”
5. 脚本会尽量按原顺序重建对应操作并恢复参数

### 4.3 顶部批量命名

1. 选中多个模型
2. 双击顶部对象名称区域
3. 输入基础名称，例如 `wall`
4. 按回车
5. 模型会被命名为 `wall_01`、`wall_02`、`wall_03`

### 4.4 一键法线贴图

1. 选中模型或材质球
2. 点击“一键法线”
3. 选择法线贴图文件
4. 工具会创建 `file` 与 `bump2d` 节点
5. 自动将贴图颜色空间设置为 `Raw`
6. 自动将法线连接到材质的 `normalCamera`

### 4.5 按 UV 壳分配随机材质 ID

1. 选中一个或多个模型
2. 点击“随机ID”
3. 工具会临时复制 UV 集、展开 UV 壳并按壳分配随机材质
4. 最后恢复当前 UV 集并清理中间历史

## 5. 功能说明

### 5.1 历史堆栈系统

相关方法：

```python
populate_stack()
select_modifier()
refresh_highlights()
toggle_node()
delete_modifier()
rename_modifier()
```

说明：

- 通过 `cmds.listHistory()` 获取当前模型历史
- 过滤材质集合、`groupParts`、`hyperLayout` 等无关节点
- 使用隔行背景色显示历史列表
- 关闭节点时通过 `nodeState` 控制是否生效
- 选中节点后自动在下方显示参数

### 5.2 Mini AE 动态参数面板

相关方法：

```python
clear_mini_ae()
populate_mini_ae()
```

说明：

- 自动收集 `keyable` 与 `channelBox` 属性
- 跳过 `message`、`caching`、`uvSet` 等不适合直接编辑的属性
- 使用 Maya 原生 `attrControlGrp` 自动生成控件
- 对没有常规参数的节点给出提示

### 5.3 历史复制与粘贴

相关方法：

```python
extract_node_data()
copy_modifiers()
paste_attributes()
paste_to_model()
```

说明：

- 复制时记录节点类型、可安全复制的属性值、组件类型
- 粘贴时尝试匹配对应 Maya 命令
- 对部分节点使用命令名映射，例如：
  - `polySmoothFace -> polySmooth`
  - `polyExtrudeFace -> polyExtrudeFacet`
  - `polyMirror -> polyMirrorFace`
- 若源节点和目标节点基础类型不一致，则跳过属性粘贴

限制说明：

- 不是所有 Maya 历史节点都能完整复刻
- 依赖选择上下文的操作在复杂场景下可能存在差异
- 生成器节点如 `polyCube`、`polySphere` 会被跳过

### 5.4 顶部对象命名区

相关方法：

```python
get_selected_transforms()
enable_rename_mode()
apply_rename()
```

说明：

- 单选对象时，顶部输入框行为等同普通重命名
- 多选对象时，工具自动循环重命名并补零编号
- 该命名只针对 transform，不直接对 shape 改名

### 5.5 材质复制与粘贴

相关方法：

```python
copy_material()
paste_material()
```

说明：

- 复制材质时记录 shading group
- 关联粘贴直接复用原 shading group
- 独立粘贴会复制材质网络后再分配

适合场景：

- 快速给多个模型套用相同材质
- 给相似模型制作独立材质副本

### 5.6 贴图检查与视图模式

相关方法：

```python
apply_map2()
spin_texture()
toggle_base_color_mode()
```

说明：

- `Map 2` 会创建一个 Lambert 检查材质网络
- 若本地存在 `C:/m341_temp/checker2.png`，会自动挂载该贴图
- `贴图 ±45°` 会查找材质历史中的 `place2dTexture` 并旋转 `rotateUV`
- `无光照 Flat` 会切换当前模型面板的 `displayLights`

### 5.7 法线贴图自动连接

相关方法：

```python
_get_materials_from_selection()
_create_normal_file_node()
_connect_normal_to_material()
quick_assign_normal_action()
```

说明：

- 自动提取当前选择对应材质
- 创建 `file` 和 `place2dTexture`
- 文件节点强制设置 `Raw` 色彩空间
- 创建 `bump2d`，并设置 `bumpInterp = 1`
- 将 `outAlpha -> bumpValue -> normalCamera` 串联起来

支持材质类型：

- `aiStandardSurface`
- `standardSurface`
- `lambert`
- `blinn`
- `phong`

### 5.8 随机材质 ID

相关方法：

```python
get_shells()
_create_random_material()
generate_mat_ids()
```

说明：

- 选中面时，直接给当前面分配一个随机材质
- 选中物体时，会按 UV 壳拆分并为每个壳分配不同材质
- 自动创建材质和 SG，并随机设置颜色

注意事项：

- 该功能会临时创建 UV 集并清理历史
- 更适合导出前制作 ID 图，而不是最终材质搭建

### 5.9 清理工具链

相关方法：

```python
clean_unused_nodes()
delete_empty_groups()
delete_all_namespaces()
```

说明：

- `清理未用`：调用 `MLdeleteUnused`
- `删空组`：清理空组、无点 mesh、无子级 transform
- `删命名空间`：递归移除用户命名空间并尝试把内容移动回根目录

风险提示：

- 这些操作具有破坏性
- 使用前建议保存场景或手动备份

### 5.10 智能选内壳

相关方法：

```python
_get_world_matrix()
_get_local_bbox()
_get_global_bbox_center()
_is_point_inside_bbox()
_get_size()
select_inner_parts()
```

说明：

- 按世界包围盒尺寸对所选物体排序
- 判断较小物体中心点是否落在较大物体包围盒内部
- 若在内部，则视为“内壳零件”并选中

适合场景：

- 枪械内管与外壳分离
- 机械模型内部零件筛选
- 清理被包裹的小部件

局限：

- 基于包围盒中心点判断，不是严格体积布尔关系
- 对镂空、弯曲或形状复杂模型可能出现误判

## 6. 技术实现摘要

### 6.1 UI 架构

- 使用 `workspaceControl` 创建浮动工作区窗口
- 主框架已改为可拉伸布局
- 历史区和 Mini AE 区使用 `paneLayout` 分割
- 当前版本已解决窗口无法自由缩放的问题

### 6.2 选择监听

- 使用 `scriptJob(event=["SelectionChanged", ...])` 监听选择变化
- 当用户切换对象时自动刷新历史列表和参数区

### 6.3 数据复制策略

- 属性复制只处理安全类型
- 自动避开连接输入、数组属性、底层 mesh 数据属性
- 通过黑名单和类型筛选降低报错概率

### 6.4 安装方式

脚本包含拖拽安装入口：

```python
onMayaDroppedPythonFile(*args)
```

实现内容：

- 自动把脚本复制到 Maya `scripts` 目录
- 自动在当前 Shelf 上创建或更新按钮
- 自动执行 `run()` 打开工具

## 8. 使用建议

- 适合在建模中后期作为常驻辅助窗口使用
- 粘贴历史前，尽量保证目标模型拓扑类型和选择上下文相近
- 清理类命令执行前先保存场景
- 批量命名适合配合多物体整理使用
- 法线贴图连接功能适合快速搭建预览材质，不代替完整材质规范检查

---

## 9. 已知限制

- 历史复刻并非对所有节点都 100% 通用
- `delete_all_namespaces()` 中使用了 `unicode()`，在 Python 3 环境下异常路径可能存在兼容性风险
- `Map 2` 默认依赖本地路径 `C:/m341_temp/checker2.png`
- 智能选内壳基于包围盒中心判断，复杂模型可能误判
- 工具按钮区域仍有部分固定列宽，窗口缩得过窄时可能拥挤

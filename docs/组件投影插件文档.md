# 组件投影插件文档

> 适用范围：Maya 2022+
> 插件目标：将选中的模型组件（顶点、边、面）实时投影到目标表面，支持平滑、膨胀、偏移等参数调整
> 当前脚本：`脚本/建模工具/组件投影插件0.2.py`

---

## 1. 功能概述

组件投影插件是一个用于将选中的模型组件实时吸附到目标表面的 Maya 工具。它支持多种投影方式和参数调整，可以快速对齐和贴合模型，解决建模中常见的穿插问题。

### 1.1 核心功能

- 实时投影：选择组件后，拖动模型时会自动吸附到目标表面
- 对象轴向投影：沿指定的 RGB 向量方向发射射线
- 基础法线投影：沿目标表面法线方向投影
- 目标平滑：临时细分目标表面，使投影更圆滑
- 目标膨胀：将目标表面沿法线方向扩展或收缩
- 最终吸附偏移：投影完成后沿法线额外偏移，避免穿插
- 反向投影方向：翻转当前射线方向
- 双向射线探测：同时向正反方向发射，自动选择最近表面
- 暂停/重置：支持暂停更新和重置状态

### 1.2 适用场景

- 模型贴合：将一个模型的表面贴合到另一个模型的表面
- 纠正穿插：解决模型之间的穿插问题
- 快速对齐：快速将组件对齐到指定表面
- 拓扑修改后：修改拓扑后可以快速重新吸附

## 2. 使用方法

### 2.1 基本工作流程

1. **设定目标**：在场景中选择作为吸附表面的模型，点击「设定吸附目标 / 基底」
2. **选择组件**：选择需要吸附的模型顶点、边或面
3. **开启吸附**：点击「开始实时投影吸附」按钮开启 Live 模式
4. **调整参数**：根据需要调整投影参数
5. **移动模型**：拖动模型，选中的组件会实时吸附到目标表面

### 2.2 UI 界面说明

UI 窗口尺寸固定为 `240 × 395`，从上到下依次为：

| 区域 | 说明 |
|------|------|
| 目标区域 | 显示当前设定的目标模型，点击「？」打开帮助文档 |
| 投影方式 | 下拉菜单选择「对象轴向」或「基础法线」 |
| RGB 向量 | 三个输入框设置投影方向（红X、绿Y、蓝Z） |
| 目标平滑 | 细分目标表面的程度，值越大越平滑 |
| 目标膨胀 | 将目标表面沿法线向外（正）或向内（负）扩展 |
| 最终吸附偏移 | 投影后沿法线额外偏移的距离 |
| 反向投影方向 | 勾选后翻转当前射线方向 |
| 双向射线探测 | 同时向正反方向发射，自动选择最近表面 |
| 实时状态控制 | 暂停/继续、重置状态 |
| 主按钮 | 开始实时投影吸附 |

### 2.3 操作示例

#### 示例 1：向下投影贴合地面

1. 选择地面模型，点击「设定吸附目标 / 基底」
2. 选择需要贴合的模型的底面顶点
3. 确认 RGB 向量为 `0, -1, 0`（向下）
4. 点击「开始实时投影吸附」
5. 向下移动模型，顶点会贴合地面

#### 示例 2：使用偏移避免穿插

1. 设定目标模型
2. 选择需要吸附的组件
3. 设置「最终吸附偏移」为一个较小的正值（如 `0.1`）
4. 开启吸附
5. 组件会贴合目标表面并向外偏移，避免穿插

#### 示例 3：修改拓扑后继续

1. 在吸附状态下点击「暂停」
2. 修改模型拓扑（加线、挤出等）
3. 重新选择需要吸附的顶点
4. 再次点击「开始实时投影吸附」恢复吸附

## 3. 参数详解

### 3.1 投影对齐方式

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| 对象轴向 | 沿指定的 RGB 向量方向发射射线 | 有明确的投影方向，如向下、向前 |
| 基础法线 | 沿目标表面的法线方向投影 | 需要贴合到任意朝向的表面 |

### 3.2 RGB 向量

用于设置「对象轴向」模式下的投影方向：

- `X`：红，左右方向
- `Y`：绿，上下方向（默认 `-1` 向下）
- `Z`：蓝，前后方向

常用方向：

| 方向 | X | Y | Z |
|------|---|---|---|
| 向上 | 0 | 1 | 0 |
| 向下 | 0 | -1 | 0 |
| 向前 | 0 | 0 | 1 |
| 向后 | 0 | 0 | -1 |
| 向右 | 1 | 0 | 0 |
| 向左 | -1 | 0 | 0 |

### 3.3 目标平滑

- **作用**：在内存中临时细分目标模型，使投影表面更圆滑
- **单位**：细分次数
- **建议值**：`0-2`，过大可能影响性能
- **注意**：平滑只在投影时生效，不会修改原目标模型

### 3.4 目标膨胀

- **作用**：将目标表面沿法线方向扩展或收缩后再投影
- **单位**： Maya 单位
- **正值**：向外扩展
- **负值**：向内收缩
- **适用场景**：需要投影到稍微高或低的位置

### 3.5 最终吸附偏移

- **作用**：投影完成后，沿表面法线额外偏移的距离
- **单位**：Maya 单位
- **常用场景**：
  - 正值：避免模型穿插或 Z-Fighting
  - 负值：嵌入到表面内部

### 3.6 选项开关

| 选项 | 说明 |
|------|------|
| 反向投影方向 | 翻转当前设定的射线方向 |
| 双向射线探测 | 同时向正反方向发射，自动选择距离最近的表面 |

## 4. 技术实现

### 4.1 核心类

```python
class ProjectComponentTool:
    def __init__(self):
        self.window_name = "projectCompUI_vFinal"
        self.help_window_name = "projectCompHelpUI"
        self.target_mesh = None
        self.temp_target_mesh = None
        self.cache = {}          # 缓存状态
        self.callback_ids = []   # 回调 ID 列表
        self.is_paused = False   # 暂停标记
        self.is_updating = False # 更新中标记
```

### 4.2 主要方法

| 方法 | 说明 |
|------|------|
| `create_ui()` | 创建 UI 窗口 |
| `show_tutorial()` | 显示帮助文档 |
| `set_target()` | 设定吸附目标 |
| `toggle_pause()` | 切换暂停/继续状态 |
| `reset_state()` | 重置状态 |
| `start_projection_session()` | 开始实时投影 |
| `rebuild_temp_target()` | 重建临时目标（平滑/膨胀） |
| `update_projection()` | 更新投影状态 |

### 4.3 投影算法流程

1. 读取当前参数（射线方向、偏移等）
2. 获取世界矩阵和逆矩阵
3. 将局部射线方向转换为世界空间
4. 遍历选中的顶点：
   - 将顶点位置转换到世界空间
   - 发射射线检测与目标表面的交点
   - 双向检测时比较正反方向的距离
   - 计算最终吸附位置（加偏移）
5. 将新位置应用到模型
6. 刷新视图

### 4.4 回调机制

使用 `OpenMaya.MNodeMessage.addNodeDirtyCallback()` 监听模型变化：

```python
self.callback_ids.append(
    om.MNodeMessage.addNodeDirtyCallback(m_obj, self.api_callback_wrapper)
)
```

当模型变脏时触发 `update_projection()`，实现实时更新。

## 5. 注意事项

### 5.1 拓扑修改

- 修改拓扑（加线、挤出、删除等）会改变顶点数量
- 检测到拓扑变化时会自动暂停
- 需要重新选择顶点并再次点击「开始实时投影吸附」

### 5.2 性能建议

- 目标平滑值不要过大（建议不超过 `2`）
- 双向射线探测会增加计算量，单方向足够时可关闭
- 选择的组件数量过多可能影响性能

### 5.3 Undo/Redo

- 插件使用 `MFnMesh.setPoints()` 修改顶点位置
- 操作会自动记录在 Maya 的 Undo 队列中
- 可以使用 `Ctrl+Z` 回退，`Ctrl+Y` 重做

### 5.4 临时目标

- 平滑和膨胀会创建临时目标模型
- 临时模型会自动隐藏
- 窗口关闭或重置时会自动删除临时目标

## 6. 常见问题

### 6.1 组件没有吸附

**可能原因**：
- 没有先设定目标模型
- 没有选择组件就点击开始
- 射线方向没有指向目标表面

**解决方法**：
1. 确认目标已设定
2. 确认已选择顶点/边/面
3. 检查射线方向是否正确
4. 尝试开启「双向射线探测」

### 6.2 吸附位置不对

**可能原因**：
- 射线方向反了
- 目标膨胀或偏移设置不合理
- 没有选择正确的投影方式

**解决方法**：
- 勾选/取消「反向投影方向」
- 调整目标膨胀和最终偏移值
- 尝试切换投影方式

### 6.3 修改拓扑后无法继续

**解决方法**：
1. 点击「暂停」
2. 修改拓扑
3. 重新选择顶点
4. 再次点击「开始实时投影吸附」

### 6.4 性能卡顿

**解决方法**：
- 减少目标平滑值
- 关闭「双向射线探测」（如果不需要）
- 减少选中的组件数量

## 10. 维护建议

### 10.1 扩展功能

如果需要扩展功能，建议：

1. 在 `create_ui()` 中添加新控件
2. 在 `update_projection()` 中添加新逻辑
3. 保持参数读取逻辑统一

### 10.2 性能优化

- 高频计算优先使用 `maya.api.OpenMaya`
- 缓存可复用的计算结果
- 避免在回调中做耗时操作

### 10.3 兼容性

- 当前支持 Maya 2022+
- 使用 `maya.api.OpenMaya`（API 2.0）
- UI 使用 `maya.cmds` 原生控件

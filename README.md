# MayaScriptTools

Maya Python 脚本工具集合。

## 当前内容

- `脚本/相机工具/camera_align_one_click_standalone_final.py`：相机对齐相关工具脚本。
- `脚本/建模工具/intersectionSolver/intersectionSolver.py`：模型穿插检测与辅助修复工具。
- `脚本/建模工具/小学智修改器历史窗口V14/小学智修改器历史窗口 V14.py`：修改器历史窗口旧版本脚本。
- `脚本/建模工具/小学智修改器历史窗口16/小学智修改器历史窗口16.py`：修改器历史窗口当前版本脚本。
- `脚本/建模工具/小学智晶格分段修改器V2.3/小学智晶格分段修改器 V2.3.py`：晶格分段修改器脚本。
- `脚本/建模工具/组件投影插件0.2/组件投影插件0.2.py`：组件投影插件脚本。
- `脚本/建模工具/glb工具maya/`：Maya GLB 导入导出插件。
- `docs/`：脚本规范、重构与修复记录。

## 使用方式

根据 `docs/Maya_Python脚本编写规范.md` 中的规范，在 Maya Python 环境中运行或集成对应脚本。

## 说明

仓库不会提交 Python 缓存、虚拟环境、编辑器配置等本地生成文件。
